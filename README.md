# Book Library App

A simple Book Library Web Application built using HTML, CSS, and JavaScript.
This app allows users to add, view, update, and delete books with a clean and interactive UI.

# Technologies Used

HTML5

CSS3

JavaScript

# How It Works

-User enters book details in the form.

-A constructor function creates a new book object.

-Each book is assigned a unique ID.

-The book is stored in an array (myLibrary).

# Key Concepts Learned

-JavaScript Constructor Functions

-Arrays and Objects

-DOM Manipulation

-Event Handling

-Unique ID generation

-Basic OOP concepts
